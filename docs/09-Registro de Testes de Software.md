# Registro de Testes de Software

Relatório com as evidências dos testes de software realizados na aplicação pela equipe, baseado no plano de testes pré-definido.

Os resultados dos testes funcionais realizados na aplicação são descritos a seguir.

## Caso de Teste 1
|Caso de Teste    | CT-01 - Cadastro de credenciais|
|:---|:---|
| Objetivo |Verificar se o processo de registro de novos usuários está funcionando corretamente, permitindo que os usuários preencham os campos necessários. |
| Passos| 1) Acessar o site <br/> 2) Clicar no botão "Criar uma conta" <br/> 3) Inserir todos os dados requisitados para cadastro <br/> 4) Clicar no botão "Salvar" <br/> 5) Em seguida clicar no botão "Fazer login "Na página direcionada pelo site <br/> 6) clicar no campo e-mail adress e inserir o e-mail cadastrado <br/> 7) Clicar no campo password e inserir a senha cadastrada <br/> 8) Clicar no botão "Entrar no meu LEER"|
| Resultados obtidos | -É possível cadastrar um novo usuário <br/> -Os campos de nome, e-mail, senha e confirmação de senha foram preenchidos corretamente.|

## Caso de Teste 3

|Caso de Teste    | CT-03 - Criar, editar e excluir um novo tema|
|:---|:---|
| Objetivo| Verificar se a função de temas está operando corretamente, possibilitando aos usuários a inclusão, edição, e exclusão de temas para organizar seus estudos.|
| Passos| 1) Na página inicial ao lado esquerdo na coluna Temas clicar no botão "Adicionar tema"<br/> 2) Clicar no campo Digite o novo tema<br/> 3) Digitar o nome escolhido<br/> 4) Clicar no botão ok para salvar<br/> 5) Clicar no icone editar, inserir novo texto para o tema e clicar em ok para salvar<br/> 6) Para excluir clicar no icone X e clicar em ok para mensagem "Tem certeza que deseja excluir este tema?"<br/> |
| Resultados obtidos | -A lista de temas foi atualizada de forma correta após a inclusão de novas categorias de temas. <br/> -A funcionalidade de edição permitiu alterar o nome corretamente. <br/> -A funcionalidade de deleção permitiu remover os temas com sucesso.|
